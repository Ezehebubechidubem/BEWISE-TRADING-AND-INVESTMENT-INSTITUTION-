Notes & Tips

  • To lock the entire app to your device: compute your device fingerprint (see instructions below) and set the value as ALLOWED_DEVICE_HASH in Render environment variables.
  • Files are stored in /uploads and the SQLite DB is data.db. For production use move to Postgres + object storage.
Device fingerprint JS (run on your phone's console to get it): 
(async ()=>{
  let uuid = localStorage.getItem('training_device_uuid');
  if(!uuid){ uuid = crypto.randomUUID ? crypto.randomUUID() : ('u-'+Math.random().toString(36).slice(2)); localStorage.setItem('training_device_uuid', uuid); }
  const data = [navigator.userAgent||'', navigator.platform||'', (navigator.languages||[]).join(','), screen.width+'x'+screen.height+'x'+screen.colorDepth, Intl.DateTimeFormat().resolvedOptions().timeZone||'', uuid].join('||');
  const enc = new TextEncoder().encode(data);
  const hashBuf = await crypto.subtle.digest('SHA-256', enc);
  const fp = Array.from(new Uint8Array(hashBuf)).map(b=>b.toString(16).padStart(2,'0')).join('');
  console.log("device fingerprint:", fp);
  alert("device fingerprint: " + fp);
})();
